# Assets Skill

## Purpose
Generate channel-ready packaging assets from a final script.

## Inputs Required
- Final script
- Channel style notes
- Brand voice rules

## Workflow
1. Generate title options.
2. Generate thumbnail concepts.
3. Generate description draft.
4. Generate social post variants.

## Output Format
- 5 Title Options
- 3 Thumbnail Concepts
- YouTube Description
- X Post Draft

## Quality Checklist
- [ ] Message consistency across assets
- [ ] Clear audience targeting
- [ ] Strong CTA language
