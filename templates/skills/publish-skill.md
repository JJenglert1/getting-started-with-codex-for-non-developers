# Publish Skill

## Purpose
Prepare final assets for publication and handoff.

## Inputs Required
- Approved script
- Approved packaging assets
- Publishing date

## Workflow
1. Run final consistency check.
2. Organize assets in publish folder.
3. Create final publishing checklist.
4. Generate follow-up promo copy.

## Output Format
- Publish-ready asset list
- Final checklist
- Post-publish follow-up copy

## Quality Checklist
- [ ] All required assets present
- [ ] Naming and organization clean
- [ ] CTA and links included
