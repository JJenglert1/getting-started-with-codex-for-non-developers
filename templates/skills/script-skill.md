# Script Skill

## Purpose
Turn a single topic brief into a clear, high-retention script draft.

## Inputs Required
- Topic + promise
- Target audience
- Desired length
- CTA

## Workflow
1. Draft hook options.
2. Build narrative arc.
3. Write section-by-section script.
4. Add transitions and CTA.

## Output Format
- Hook
- Section Outline
- Full Script Draft
- CTA Block

## Quality Checklist
- [ ] Hook is specific and useful
- [ ] Flow is easy to follow
- [ ] No unnecessary fluff
