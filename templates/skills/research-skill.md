# Research Skill

## Purpose
Use this skill for fast market/competitor/content research.

## Inputs Required
- Topic
- 3-5 references or source links
- Target audience

## Workflow
1. Extract key claims and patterns from each source.
2. Cluster insights into themes.
3. Identify opportunities/gaps.
4. Recommend next actions.

## Output Format
- Executive Summary (5-7 bullets)
- Pattern Table (hook, structure, angle)
- Gap Analysis
- Recommended Next Moves

## Quality Checklist
- [ ] Distinguishes facts from inference
- [ ] Includes concrete examples
- [ ] Provides actionable recommendations
