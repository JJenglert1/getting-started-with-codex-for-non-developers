# MCP Setup Checklist

## Core Setup
- [ ] Identify the 2-3 tools you use every day
- [ ] Connect those MCP servers first
- [ ] Verify basic read access works
- [ ] Verify any write actions require confirmation if needed

## Suggested First MCP Connections
- [ ] Calendar
- [ ] Docs/Knowledge base
- [ ] Project/task tracking

## Validation Tasks
- [ ] "Summarize my calendar for tomorrow"
- [ ] "Pull top priorities from my project tracker"
- [ ] "Draft an update based on today\'s work items"

## Safety
- [ ] Confirm permission settings are correct
- [ ] Avoid broad write permissions at first
- [ ] Expand access only after successful test runs
